Setup¶

1. Package Installation¶

Python, pip, and virtualenv must be installed manually. All other packages can be installed automatically with pip. After installing Python and pip, virtualenv can be installed by:

$ sudo apt-get install python-pip
$ sudo pip install virtualenv

Virtual Environment Setup¶

Create the virtual environment and enter into it:

$ virtualenv atm-env
(atm-env) $

Install the required packages:

(atm-env) $ pip install -r requirements.txt

The required packages are:

sqlalchemy==1.1.14
numpy==1.13.1
boto==2.48.0
pandas==0.20.3
scikit-learn==0.18.2
scipy==0.19.1
sklearn-pandas==1.5.0
nolearn==0.6.0
mysql-python==1.2.5
pyyaml
-e git+https://github.com/hdi-project/btb.git#egg=btb
-e .

2. Database Setup¶

On Ubuntu, SQLite3 can be installed with the command:

$ sudo apt-get install sqlite3

Alternatively, MySQL can be installed with the command:

$ sudo apt-get install mysql-server mysql-client libmysqlclient-dev

and following the instructions.
